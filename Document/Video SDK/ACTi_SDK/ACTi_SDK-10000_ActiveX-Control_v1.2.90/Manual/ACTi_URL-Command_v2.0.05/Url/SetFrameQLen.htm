SET_FRAME_Q_LEN

Description

Set the maximum of the video frame queue length.

This is for test only and should be used internally.

Input Argument

1~4

Attribute

  • Write Only
  • On-Fly

Return of Message

SET_FRAME_Q_LEN='3'

Applicability

  • EPL Encoder Devices

Status

Valid

See Also

NETWORKACCESS_STOP, RS232_TEST, AUDIO_LOOP_BACK

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&SET_FRAME_Q_LEN=3
