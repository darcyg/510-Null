URL CGI Program

This CGI provides the URL command to access the Decoder devices.

Only root could set the configuration through this CGI. If the logon user is not root, all write commands will be treated as read commands.

The return message of URL does not follow the Specification of Return Message.

The document was created far ago and was not maintained anymore. Please use URLs in DECODER CGI for your applications.

NOTE:

This CGI will be replaced with the DECODER CGI. That will be claimed with the release of device's firmware in the future.

List of URLs

The URL command in this CGI was not maintained anymore. Please the URL command in URL.CGI for further information. For New design, please refer the URL command in DECODER CGI.

The syntax of the command is listed as follow:

http://<IP Address>/cgi-bin/url.cgi?USER=<Account Name>&PWD=<Password>&<Parameters>
