LAN_IP, LAN_NETMASK

Description

Set IP or Netmask address of LAN port.

Input Argument

IPv4 Addressing (xxx.xxx.xxx.xxx)

Attribute

  • Write Only
  • Not On-Fly

Return of Message

None

Applicability

  • WPL Decoder Devices

Status

Valid

See Also

HOST(only available in WPL Decoder Devices), LAN_HOSTNAME

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&LAN_IP=192.168.1.254
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&LAN_NETMASK=255.255.255.0
