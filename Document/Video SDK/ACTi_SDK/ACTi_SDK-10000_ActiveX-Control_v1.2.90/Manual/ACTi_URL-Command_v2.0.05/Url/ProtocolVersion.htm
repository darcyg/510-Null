PROTOCOL_VERSION

Description

Set the streaming protocol version to connect to encoder.

There are two video/audio streaming protocols we offered. One is TCP 1.0 which is out of fashion and available in all 1CH, 2CH and QUAD Encoders and Decoder in WPL Platform. The other one is TCP 2.0 which is most use. In all EPL, PlatformT and Multi-Channel devices, only this protocol is available.

Input Argument

1 : TCP 1.0
2 : TCP 2.0

Attribute

  • Write Only
  • On-Fly

Return of Message

None

Applicability

  • WPL Decoder Devices

Status

Valid

See Also

V2_STREAMING_TYPE

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&PROTOCOL_VERSION=2
