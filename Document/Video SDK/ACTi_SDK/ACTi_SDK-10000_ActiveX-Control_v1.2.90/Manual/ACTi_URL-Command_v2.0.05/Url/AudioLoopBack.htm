AUDIO_LOOP_BACK

Description

Enable or Disable the Loopback function which routes the AudioIn to AudioOut in firmware side.

This is for test only and should be used internally.

Input Argument

0: Disable, 1: Enable

Attribute

  • Write Only
  • On-Fly

Return of Message

AUDIO_LOOP_BACK='0'

Applicability

  • EPL Encoder Devices

Status

Valid

See Also

NETWORKACCESS_STOP, SET_FRAME_Q_LEN, RS232_TEST

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&AUDIO_LOOP_BACK=0
