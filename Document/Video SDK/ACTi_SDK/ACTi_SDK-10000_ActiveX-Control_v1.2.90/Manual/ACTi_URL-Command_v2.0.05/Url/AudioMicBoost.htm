AUDIO_MIC_BOOST

Description

Set the AudioIn Sensitivity configuration. Check the device specification is needed before use this command. Only two-way audio device supports this URL.

There are two levels for this volume control. They are 0dB and 20dB. Here is the mapping table of input argument and these four volume levels.

Value of Input argument Volume Level
0 0dB
1 20dB

Input Argument

0 or 1

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
AUDIO_MIC_BOOST='1'
  • Write Operation
OK: AUDIO_MIC_BOOST='1'
or
ERROR: AUDIO_MIC_BOOST='1'

Applicability

  • EPL Decoder Devices

Status

Valid

See Also

AUDIO_SET_VOL

Example

  • Read Operation

http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&AUDIO_MIC_BOOST

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&AUDIO_MIC_BOOST=1
