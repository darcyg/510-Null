DDNS_TYPE

Description

Get or Set DDNS (Dynamic Domain Name Server) type configuration.

In EPL decoder, this URL is same as the DDNS_TYPE in the SYSTEM CGI but only Type1 URL command is supported.

Therefore, the DDNS_TYPE URL in WPL decoders is only addressed in this page.

Input Argument

1: Disable
2: Enable

Attribute

  • Write Only
  • Not On-Fly

Return of Message

None

Applicability

  • WPL Decoder Devices
  • EPL Decoder Devices (refer to DDNS_TYPE in the SYSTEM CGI)

Status

Valid

See Also

DDNS(Only available in EPL Decoder Device), DDNS_HOSTNAME, DDNS_SERVICE, DDNS_USERNAME, DDNS_PASSWORD

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&DDNS_TYPE=1
