AUDIO_SET_VOL

Description

Set the AudioOut volume configuration. Check the device specification is needed before use this command. Only audio device supports this URL.

Input Argument

0~100

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
AUDIO_SET_VOL='84'
  • Write Operation
OK: AUDIO_SET_VOL='84'
or
ERROR: AUDIO_SET_VOL='84'

Applicability

  • EPL Decoder Devices

Status

Valid

See Also

AUDIO_MIC_BOOST

Example

  • Read Operation

http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&AUDIO_SET_VOL

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&AUDIO_SET_VOL=84
