HOST

Description

Get or all main settings on the devices. These settings include

LAN_HOSTNAME, LANGUAGE, LAN_IP, LAN_HOSTNAME, BAUDRATE, UARTSETTING, OSD_X, OSD_Y, VIDEO_TV, VIDEO_OSD, VOL_AUDIOOUT and VOL_AUDIOIN.

Input Argument

None

Attribute

  • Read Only

Return of Message

LAN_HOSTNAME='Test'
LANGUAGE='0'
LAN_IP='100.100.100.1'
LAN_NETMASK='255.255.255.0'
BAUDRATE='9600'
UARTSETTING='1'
OSD_X=''
OSD_Y='50'
VIDEO_TV='PAL'
VIDEO_OSD=''
VOL_AUDIOOUT=''
VOL_AUDIOIN='2'

Applicability

  • WPL Decoder Devices

Status

Valid
