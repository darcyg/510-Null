NETWORKACCESS_STOP

Description

Stop the networking thread in the decoder and show the color bar on the TV Out.

This is for test only and should be used internally.

Input Argument

None

Attribute

  • Write Only
  • On-Fly

Return of Message

NETWORKACCESS_STOP

Applicability

  • EPL Encoder Devices

Status

Valid
