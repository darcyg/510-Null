DDNS_HOSTNAME

Description

Get or Set DDNS (Dynamic Domain Name Server) host name configuration. This allows people to access this device through this host name.

In EPL decoder, this URL is same as the DDNS_HOSTNAME in the SYSTEM CGI but only Type1 URL command is supported.

Therefore, the DDNS_HOSTNAME URL in WPL decoders is only addressed in this page.

Input Argument

Character strings (< 128 Bytes)

Attribute

  • Write Only
  • Not On-Fly

Return of Message

None

Applicability

  • WPL Decoder Devices
  • EPL Decoder Devices (refer to DDNS_HOSTNAME in the SYSTEM CGI)

Status

Valid

See Also

DDNS(Only available in EPL Decoder Device), DDNS_TYPE, DDNS_SERVICE, DDNS_USERNAME, DDNS_PASSWORD

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&DDNS_HOSTNAME=camera1.com
