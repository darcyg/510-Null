RS232_TEST

Description

Loopback the Serial Port on the firmware side. This will echo the input data from the RXD in Serial Port to TXD in Serial Port by firmware.

This is for test only and should be used internally.

Input Argument

None

Attribute

  • Write Only
  • On-Fly

Return of Message

None

Applicability

  • EPL Encoder Devices

Status

Valid

See Also

NETWORKACCESS_STOP, SET_FRAME_Q_LEN, AUDIO_LOOP_BACK
