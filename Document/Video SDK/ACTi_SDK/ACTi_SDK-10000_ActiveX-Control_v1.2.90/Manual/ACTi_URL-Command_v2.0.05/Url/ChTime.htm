CHxx_TIME

Description

Set the time of stay in second for a connected channel to the decoder.

In the WPL decoder devices, it could connect an encoder and move to next channel after a period of time. This URL sets how long the decoder stays this channel before moves to next channel. If the time is set to 0, the decoder will stay this channel forever.

Input Argument

0~65535

Attribute

  • Write Only in WPL Decoders
  • Not On-Fly

Return of Message

None

Applicability

  • WPL Decoder Devices

Status

Valid

See Also

CHxx_VIDEO_CONNECT, CHxx_VIDEO_STREAM, CHxx_VIDEO_AUDIO, CHxx_VIDEO_CONTROL, CHxx_VIDEO_TCP_IP, CHxx_VIDEO_MULTICAST_IP, CHxx_VIDEO_USERNAME, CHxx_VIDEO_PASSWORD, CCHxx_PORT_STREAM, CHxx_PORT_CONTROL, CHxx_PORT_MULTICAST, CHxx_PORT_RTP, CHxx_PORT_HTTP, CHxx_SOURCE_TYPE, CHxx_QUAD_CHANNEL, CHxx_CHANNEL, SWITCH, VIDEO_VARFPS

Example

  • Write Operation
http://ip:port/cgi-bin/url.cgi?USER=Admin&PWD=123456&CHxx_TIME=1
where xx is from 01~16
