QUAD CGI Program

Only root could set the configuration through this CGI. If the logon user is not root, all write commands will be treated as read commands.

The CGI is only available in the WPL QUAD Encoder Devices.

In QUAD Encoder, it could show 4 channels' video in one single video stream (QUAD Mode) or show one of four channels in one single video stream (Channel Mode).

There is only Type1 URL command supported in this CGI.

The format of the return message does not follow the specification of return message. The return message will be described in every URL command.
