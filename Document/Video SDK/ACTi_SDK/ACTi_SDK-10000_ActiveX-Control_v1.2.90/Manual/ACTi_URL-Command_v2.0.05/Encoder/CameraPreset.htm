CAMERA_PRESET

Description

Reset the Sensor Module or the analog PTZ camera.

This URL will reset the sensor module of an analog PTZ camera to the factory default configuration.

For non-PTZ IP camera, the return message will be "ERROR: CAMERA_PRESET not found".

Input Argument

None

Attribute

  • Write Only
  • On-Fly

Return of Message

  • Write Operation
OK
or
ERROR: CAMERA_PRESET not found'

Applicability

  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders

Status

Valid

See Also

None

Example

  • Write Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&CAMERA_PRESET
or
http://ip:port/cgi-bin/cmd/encoder?CAMERA_PRESET
