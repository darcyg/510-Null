REBOOT

Description

Reboot the device. The runtime configuration might be lost if it has not be saved into flash after this URL was executed

Input Argument

None

Attribute

  • Not On-Fly

Return of Message

Always return OK

Applicability

  • WPL Platform : All Devices
  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

FACTORY_DEFUALT,  SAVE_REBOOT, SAVE

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&REBOOT
or
http://ip:port/cgi-bin/cmd/system?REBOOT
