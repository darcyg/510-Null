DNS

Description

Get the DNS (Domain Name Server) configurations.

If the CHANNEL=n was set with this URL to the Multi-Channel device, the Router's DNS configurations will be returned.

Input Argument

None

Attribute

  • Read Only

Return of Message

DNS_PRIMARY='168.95.1.1'
DNS_SECONDARY=''

Applicability

  • WPL Platform : All Devices
  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

DNS_PRIMARY, DNS_SECONDARY, WAN_STATUS, V2_WAN_STATUS

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&DNS
or
http://ip:port/cgi-bin/cmd/system?DNS

HOME, SYSTEM CGI
