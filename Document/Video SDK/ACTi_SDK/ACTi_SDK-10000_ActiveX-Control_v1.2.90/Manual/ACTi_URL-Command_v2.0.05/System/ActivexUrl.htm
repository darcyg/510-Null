ACTIVEX_URL

Description

Read the ActiveX URL configuration.

Currently, the ActiveX URL setting is not used in the device. However, for some applications, the remote host needs to read this setting.

If the CHANNEL=n was set with this URL to the Multi-Channel device, the Router's ACTIVEX_URL configurations will be returned.

Input Argument

None

Attribute

  • Read Only

Return of Message

  • Read Operation
ACTIVEX_URL=''

Applicability

  • WPL Platform : All Encoder Devices
  • EPL Platform : All Encoder Devices
  • PlatfotmT : All Encoder Devices
  • Multi-Channel Encoder : Router Only

Status

Valid

See Also

Example

  • Read Operation

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&ACTIVEX_URL
or
http://ip:port/cgi-bin/cmd/system?ACTIVEX_URL
