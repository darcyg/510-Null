TOS

Description

Get the ToS(Type of Service) configurations. The ToS setting will be set to the ToS field in IP header of every video and audio packet.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the CHANNLE 1 video server's TOS configurations will be returned. If the CHANNEL 1 video server does not connect to the router, the error message will be returned.

Input Argument

None

Attribute

  • Read Only

Return of Message

VIDEO_TOS_TYPE='1'
VIDEO_TOS_PRIORITY='Normal-Service'

Applicability

  • WPL Platform : All Encoder Devices
  • EPL Platform : All Encoder Devices
  • PlatfotmT : All Encoder Devices
  • Multi-Channel Encoder : All Devices

Status

Valid but will be obsolete in the future. That will be claimed with the release of device's firmware.

See Also

VIDEO_TOS_TYPE, VIDEO_TOS_PRIORITY

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&TOS
or
http://ip:port/cgi-bin/cmd/system?TOS
