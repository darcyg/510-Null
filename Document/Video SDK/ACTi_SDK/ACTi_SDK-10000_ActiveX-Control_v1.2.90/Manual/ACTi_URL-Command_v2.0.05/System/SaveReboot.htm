SAVE_REBOOT

Description

Save the configuration to the flash and reboot the device.

Input Argument

None

Attribute

  • Not On-Fly

Return of Message

Always return OK

Applicability

  • WPL Platform : All Devices
  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

FACTORY_DEFUALT,  SAVE, REBOOT

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&SAVE_REBOOT
or
http://ip:port/cgi-bin/cmd/system?SAVE_REBOOT
