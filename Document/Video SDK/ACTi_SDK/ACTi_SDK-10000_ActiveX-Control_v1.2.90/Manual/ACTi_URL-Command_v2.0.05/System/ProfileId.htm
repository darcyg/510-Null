PROFILE_ID

Description

Get device's camera profile ID.

IThe camera profile managements the capability and configuration of the camera front-end. The remote host could know what the sensor is used in the device by looking at this profile ID. The coding of the Profile ID is not in this document. Please refer the profile naming rule document for that.

This command is only available in all encoders in EPL, PlatformT devices and 4CH, 16CH Multi-Channel Encoders.

Input Argument

None

Attribute

  • Read Only

Return of Message

  • Read Operation
PROFILE_ID='MT9M131-RA0-V071214A'

Applicability

  • EPL Platform : All Encoder Devices
  • PlatformT : All Encoder Devices
  • Multi-Channel Encoder : 4CH and 16CH Devices

Status

Valid
