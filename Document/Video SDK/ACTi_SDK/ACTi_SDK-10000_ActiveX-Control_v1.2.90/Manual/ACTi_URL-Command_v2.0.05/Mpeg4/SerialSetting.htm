SERIAL_SETTING

Description

Get or Set the serial port configuration.

There is a serial port on the device for the RS232/RS485/RS422 port. The external device could be accessed or controlled through this port. For example, when the RS485 port in Analog PTZ camera connects to the Video Server's serial port, user could control this Analog PTZ camera by the Video Server through the Ethernet at remote site.

Note that, there is no any hardware and software flow control mechanism implemented in this serial port. For example, when the RS485 port in Analog PTZ camera connects to the Video Server's serial port, user could control this Analog PTZ camera by the Video Server through the Ethernet at remote site.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, this command is read-only command and the CHANNEL 1 video server's SERIAL_SETTING configuration will be returned. If the CHANNEL 1 video server does not connect to the router, the error message will be returned.

Input Argument

n,m
where
n is the serial data configuration. It could be
8N1: 8 Data bits, No parity check, 1 Stop bit
8O1: 8 Data bits, Odd parity check, 1 Stop bit
8E1: 8 Data bits, Even parity check, 1 Stop bit
m is the baud rate. It could be 2400, 4800, 9600, 19200, 38400, 57600, 115200

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
OK: SERIAL_SETTING='8N1,9600'
  • Write Operation
OK: SERIAL_SETTING='8N1,9600'
or
ERROR: SERIAL_SETTING='8N1,9600'

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : All Video Servers

Status

Valid

See Also

SERIAL_ASCII, SERIAL_HEX

Example

  • Read Operation

http://ip:port/cgi-bin/mpeg4?USER=Admin&PWD=123456&SERIAL_SETTING
or
http://ip:port/cgi-bin/cmd/mpeg4?SERIAL_SETTING

  • Write Operation
http://ip:port/cgi-bin/mpeg4?USER=Admin&PWD=123456&SERIAL_SETTING=8N1,9600
or
http://ip:port/cgi-bin/cmd/mpeg4?SERIAL_SETTING=8N1,9600
