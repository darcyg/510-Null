VIDEO_BRIGHTNESS, VIDEO_CONTRAST, VIDEO_SATURATION, VIDEO_HUE

Description

Get or set the video parameters.

Note that VIDEO_HUE is Read-Only in some EPL and PlatformT encoders. Please check devices' specifications.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, this command is read-only command and the CHANNEL 1 video server's VIDEO_BRIGHTNESS configuration will be returned. If the CHANNEL 1 video server does not connect to the router, the error message will be returned.

Input Argument

0~100.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
OK: VIDEO_BRIGHTNESS='50'
OK: VIDEO_CONTRAST='50'
OK: VIDEO_SATURATION='50'
OK: VIDEO_HUE='50'
  • Write Operation
OK: VIDEO_BRIGHTNESS='50'
OK: VIDEO_CONTRAST='50'
OK: VIDEO_SATURATION='50'
OK: VIDEO_HUE='50'
or
ERROR: VIDEO_BRIGHTNESS='50'
ERROR: VIDEO_CONTRAST='50'
ERROR: VIDEO_SATURATION='50'
ERROR: VIDEO_HUE='50'

Applicability

  • WPL Platform : All 1CH and 2CH Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : All Video Servers

Status

Valid
