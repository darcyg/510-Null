Mpeg4 CGI Program

Only root could set the configuration through this MPEG4 CGI. If the logon user is not root, all write commands will be treated as read commands. The CGI is only available in the Encoder Devices. The URL in this CGI will be forwarded to all video servers by router in the Multi-Channel Encoder platform.

NOTE:

This CGI will be replaced with the Encoder CGI. That will be claimed with the release of device's firmware in the future.
